# 小さな勝利 - Webアプリ

開発中の「小さな勝利」記録アプリ。

## 機能

### 実装済み（MVP）

- ✅ 小さな勝利を入力・記録
- ✅ ローカルストレージに保存
- ✅ 一覧表示
- ✅ 削除機能
- ✅ 文字数カウント（最大200文字）
- ✅ 統計表示（総記録数、連続記録日数）

### 今後の機能

- カテゴリ分け（有料版）
- 週次・月次の統計（有料版）
- 全期間の記録保存（有料版）
- メモ機能（有料版）

## 技術スタック

- フロントエンド: HTML, CSS, JavaScript
- データ保存: LocalStorage
- ホスティング: GitHub Pages（予定）
